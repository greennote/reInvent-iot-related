## これは何？

re:Invent 2021では大量の発表がありましたが、その中からIoT関連のものを収集しました。[リスト](re_Invent_2021_IoT.csv)はこちらにあります。

## ご注意

* このリストは公式のものではありません。記載されている情報の正確さについては保証しません。サービスや機能の利用に当たっては、必ずAWS公式の資料を参照してください。
* re:Invent 2021の前後に発表された内容も含みます。
